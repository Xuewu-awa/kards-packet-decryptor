# Kards 数据包解密工具

## 简介

Kards 是一款二战题材的卡牌游戏，由1939 Games开发。本工具用于解密其游戏数据包，查看原始通信内容。

## 加密方式

### 数据包格式

```
[2位表索引][6位数据长度][4位动作ID(Base64)][密钥][密文(Base64)]
```

### 解密原理

1. 表索引：前2位数字，用于从固定长度表获取密钥长度
2. 密钥：随机生成的Base64字符组成的字符串
3. XOR加密：使用密钥对数据进行循环异或加密
4. 动作ID：24位整数，与密钥前3字节异或加密
5. Base64：动作ID和密文都经过Base64编码

## 使用方法

```bash
python main.py "数据包"
```
